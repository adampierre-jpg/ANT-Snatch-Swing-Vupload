# VBT System Comparison

This repository contains two versions of the VBT (Velocity-Based Training) system:

## VBT v3.5 (Full Version)
**Files**: `index.html`, `app.js`, `style.css`

### Features
- ✅ Complete movement classification (Swing, Clean, Snatch, Press)
- ✅ Re-clean detection
- ✅ Dual-hand tracking with automatic side locking
- ✅ Glycolytic fatigue tracking with color-coded warnings
- ✅ Zone-based position detection (Floor, Backswing, Rack, Overhead)
- ✅ Separate velocity baselines for cleans and presses
- ✅ Make.com webhook integration
- ✅ Advanced state machine for movement detection
- ✅ Professional coaching features

### Best For
- Essential Fitness coaching programs
- Dad Ready assessments
- Tempered programming
- Professional athlete tracking
- Complex movement analysis

---

## VBT Vanilla 50 (Simplified Version)
**Files**: `vbt-vanilla-50.html`, `vbt-vanilla-50.js`, `vbt-vanilla-50-style.css`

### Features
- ✅ Simple rep counting
- ✅ Basic velocity tracking
- ✅ Peak velocity measurement
- ✅ Rep history with timestamps
- ✅ Local JSON export
- ✅ Right-hand tracking only

### Best For
- Learning VBT concepts
- Basic velocity tracking
- Simple rep counting
- Quick testing
- Minimal overhead applications
